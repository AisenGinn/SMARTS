Overview

This competition consists of two parts, namely
(a) Track-1: Online Learning
(b) Track-2: Offline Learning

Code

(a) All necessary code for the competition is provided at SMARTS/competition directory in the SMARTS repository.
(b) To obtain the code, do the following: 

$ git clone https://github.com/huawei-noah/SMARTS.git
$ git checkout comp-4
$ cd /SMARTS/competition
